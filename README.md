# Starter App

> Fakebet v1.1

## Commands

### Development

##### Front-end

```sh
# for vue, need @vue/cli and @vue/cli-service-global -g
$ cd front && npm install
$ npm run vue
```

##### Back-end

```sh
# for node, need nodemon -g
$ cd server && npm install
$ npm run node
```

### Production

```sh
# need @vue/cli and @vue/cli-service-global -g
$ cd front && npm install
$ cd server && npm install
$ npm run vue-production
$ npm run node-production
```
