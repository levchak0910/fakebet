# Fakebet
> Site with fake bets on football

Demo: [link](https://fakebet.herokuapp.com)
Description: [link](https://fakebet.herokuapp.com/about)

## Commands
    # install dependencies
    npm install
    # development (need nodemon -g)
    npm run dev
    npm run node
    # production
    npm run build
    npm run node-clear

## Using
**Front:** Vue: router, i18n, vuex, axios
**Back:** Node, Koa
**Database:** Mongo
